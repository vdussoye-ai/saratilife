# SaratiLife

**Flowing with Intention** — A strategic life portfolio platform for high-performing professionals navigating career disruption.

*Sarati (सरति) — Sanskrit for "to flow, to move"*

## The Five Capitals Framework

SaratiLife helps professionals manage their life the way a strategist manages a portfolio — across five dimensions:

- **Financial Capital** — Runway to freedom
- **Human Capital** — Skills that compound
- **Social Capital** — Network as safety net
- **Health Capital** — Energy as currency
- **Spiritual Capital** — Purpose as compass

## Stack

- **Framework:** React 18 + Vite
- **Routing:** React Router v6
- **Hosting:** Vercel
- **Analytics:** Google Analytics (G-MKY433KWFR)
- **Styling:** Inline CSS (zero dependencies)

## Local Development

```bash
npm install
npm run dev
```

## Deploy

Connected to Vercel via GitHub. Push to `main` triggers auto-deploy.

```bash
git add .
git commit -m "your message"
git push origin main
```

## Pages

| Route | Page | Description |
|-------|------|-------------|
| `/` | Launch | Landing page with value prop, Five Capitals framework, and CTA |
| `/assessment` | Career Pivot Wizard | 7-step assessment: role, skills, financial, values, aspirations → report |
| `/capital/:id` | Capital Deep Dive | Per-capital waitlist/early access pages (financial, human, social, health, spiritual) |
